# Janick Braun | Portfolio

A modern, high-performance personal portfolio built with Next.js 15, Tailwind CSS, and Framer Motion. Featuring a clean, dark-themed aesthetic with a focus on premium typography and smooth interactions.

View the live website here: https://portfolio.janickbraun.com

## Technology Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & Custom Brand SVGs
- **Deployment**: [Vercel](https://vercel.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Key Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Content**: 
  - Real-time age calculation based on birthdate.
  - Centralized data management in `src/lib/data.ts`.
- **Custom Project Layouts**: Interactive project slider and detailed view pages.
- **Hobby Integration**: Dedicated section for personal interests with custom CTAs (e.g., Letterboxd).
- **Interactive Resume**: Unified CV component used across multiple pages.
- **Modern Navigation**: Mobile hamburger menu with smooth slide-down transitions.
